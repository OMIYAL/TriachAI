# 🚀 TriArchV5 — v2026.6.22.0657 (22 June 2026)

## 💪 Improvements

**More reliable production deployments** — We've strengthened the health checks that run every time an update ships to the app. From this release onward, changes are automatically verified to be healthy before traffic switches over, so the risk of disruption during a release is significantly reduced. If something unexpected happens post-deploy, an automatic rollback kicks in to keep the service available. (#203)

**All services now live on triarch.ai** — Production endpoints across every Triarch service (Auth, API, and the public portal) are now consistently routed through the canonical triarch.ai domain, ensuring a seamless single-origin experience for users and integrations alike.
